# Wifi-i3
This is a GUI app to connect Wifi in window manager i3 on linux

### Requirements

- python >= 3.8**.************
- NetworkManager
- nmcli
- notify-send and dunstrc file


### Instalation

- pip install -r requirements.txt
- sudo pacman -S networkmanager
- sudo pacman -S dunst
- dunst --start -conf dunctrc
